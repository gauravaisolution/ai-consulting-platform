# AI Consulting Platform

A modern consulting website built with Astro and TailwindCSS, deployed on Vercel.

## Tech Stack

- **Frontend:** Astro + TailwindCSS
- **Hosting:** Vercel
- **Forms:** Tally.so
- **Scheduling:** Cal.com

## Features

- Responsive design with dark mode
- Interactive service demos
- Lead capture integration
- Calendar booking

## Quick Start

```bash
cd dev
npm install
npm run dev
```

Open http://localhost:4321

## Project Structure

```
├── dev/           # Application source code
├── learning/      # Learning resources
└── resource/      # Reference materials
```

## Deployment

Deployed automatically via Vercel on push to `main` branch.
